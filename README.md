------------------------------------------------------------------------

## Overview

This project is an AI-powered Emotion Voice Assistant that detects user
emotions from voice input and generates intelligent, context-aware
responses using a local Large Language Model powered by Ollama.

------------------------------------------------------------------------

## Features

-   Voice input recording (microphone-based)\
-   Speech-to-text conversion\
-   Emotion detection from voice/text\
-   AI-generated responses using Ollama (local LLM)\
-   Emotion-aware reply generation\
-   Text-to-speech output\
-   Clean and interactive chat-style UI

------------------------------------------------------------------------

## Tech Stack

-   Frontend: HTML, CSS, JavaScript\
-   Backend: Flask (Python)\
-   AI Runtime: Ollama\
-   Speech Processing: SpeechRecognition / Whisper (optional)\
-   TTS: pyttsx3 / gTTS

------------------------------------------------------------------------

## Setup Instructions

1.  Install Ollama: https://ollama.com\

2.  Run model: ollama run llama3

3.  Clone repo: git clone
    https://github.com/SIDMINUL/emotion-voice-assistant.git

4.  Install dependencies: pip install -r requirements.txt

5.  Run: python app.py
